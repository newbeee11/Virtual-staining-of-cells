# Virtual Pap Staining

This repository provides code and data for the paper:

"Label-free virtual staining of cervical cytology directly from brightfield images"

## 🔧 Code
- Inference code for virtual staining
- Model architecture (DTPN)

## 📊 Data
- SSIM / PSNR results
- Statistical test samples (t-test)
- Representative examples

## 🚀 Usage

```bash
python infer.py --input ./examples --output ./results
